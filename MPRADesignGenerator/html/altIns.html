altIns {MPRADesignGenerator}R Documentation

CREATE ALTERNATIVE SEQUENCES FOR INSERTIONS

Description

Insert specified bases after base 73 into provided ref seq. Shortens sequence from both sides so total length is <= 145bp.

Usage

altIns(reference_sequence, insertion_bases)

Arguments

145bp

reference sequence, string containing bases to insert.

Value

145bp alt sequence with specified bases inserted

delBases {MPRADesignGenerator}R Documentation

GET BASES TO BE DELETED

Description

Check which bases the program will delete. Can be used to compare to the bases that should be deleted per variant input file.

Usage

delBases(reference_sequence, deletion_bases)

Arguments

reference

sequence, string containing bases to delete

Value

string containing bases that the program will delete.

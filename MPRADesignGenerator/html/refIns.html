refIns {MPRADesignGenerator}R Documentation

SHORTEN REFERENCE SEQUENCES FOR INSERTIONS

Description

Ensures ref seq for insertions has the same bases removed from the ends as alt seq.

Usage

refIns(reference_sequence, insertion_bases)

Arguments

145bp

reference sequence, string containing bases that were inserted.

Value

ref sequence with appropriate end bases removed

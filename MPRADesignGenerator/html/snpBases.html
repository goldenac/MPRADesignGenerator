snpBases {MPRADesignGenerator}R Documentation

GET BASE TO BE CHANGED

Description

Check which base the program will change. Can be used to compare to the bases that should be changed per variant input file.

Usage

snpBases(reference_sequence)

Arguments

reference

sequence

Value

string containing base that the program will change.
